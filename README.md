# Flor & Afeto — Landing Page de Floricultura

Projeto fictício criado para atividade prática da disciplina **Desenvolvimento para Internet**.

## Objetivo

Criar uma landing page realista usando **HTML + CSS**, com foco em:

- estrutura HTML semântica;
- organização visual;
- CSS externo;
- responsividade básica;
- cards de produtos;
- chamada para ação;
- formulário de contato;
- imagens locais em SVG.

## Arquivos

```text
floricultura_landing_page/
├── index.html
├── style.css
├── README.md
├── enunciado_atividade.md
└── img/
    ├── favicon.svg
    ├── hero-flores.svg
    ├── produto-buque.svg
    ├── produto-arranjo.svg
    ├── produto-cesta.svg
    └── promocao.svg
```

## Como executar

Abra o arquivo `index.html` no navegador.

## Sugestões de adaptação pelos alunos

- alterar nome da floricultura;
- trocar textos e preços fictícios;
- mudar paleta de cores;
- acrescentar novos cards;
- melhorar o formulário;
- aplicar validação com JavaScript em uma próxima etapa;
- rodar Lighthouse e corrigir acessibilidade/performance.
