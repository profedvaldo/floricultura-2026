# Atividade prática — Landing Page de Floricultura

## Contexto

Uma floricultura local deseja uma página simples para divulgar seus produtos e receber contatos de clientes interessados em buquês, arranjos, cestas e flores para eventos.

A turma deverá criar ou adaptar uma **landing page em HTML + CSS**, com aparência profissional, responsiva e organizada.

## Entrega mínima

- `index.html`;
- `style.css`;
- pasta `img`, se utilizar imagens;
- print da versão desktop;
- print da versão mobile;
- arquivo `.zip` ou link do repositório.

## Estrutura obrigatória

A página deve conter:

1. Cabeçalho com nome da floricultura e menu.
2. Seção principal com título, texto e botão.
3. Seção de produtos ou serviços com pelo menos 3 cards.
4. Seção sobre a floricultura.
5. Destaque promocional.
6. Depoimento fictício de cliente.
7. Formulário de contato visual.
8. Rodapé com dados fictícios e identificação da atividade.

## Requisitos técnicos

- Usar HTML semântico: `header`, `nav`, `main`, `section`, `article`, `footer`.
- Usar CSS externo.
- Criar layout com Flexbox ou Grid.
- Usar imagens com atributo `alt`.
- Criar botões estilizados.
- Aplicar responsividade básica.
- Organizar o código com indentação.
